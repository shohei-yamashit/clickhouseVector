## Getting Started

### Prerequisites

- docker
- jq
- curl

### Installation

Clone the repository:

```sh
git clone https://github.com/shohei-yamashit/clickhouseVector.git
```

### Running the Application
compose.yamlを元にコンテナを立ち上げたあと、shellを実行するだけです。
詳細は記事をご確認ください。

### Building the Application

特段不要です
